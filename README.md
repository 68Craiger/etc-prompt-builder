# ETC - Engineering the Conversation

A professional prompt builder for creating structured, high-quality AI prompts.

## Features

- **5 Frameworks**: Structured Prompt, Few-Shot, Chain-of-Thought, Task Decomposition, Decision Framework
- **Quality Signals**: Real-time feedback on prompt quality
- **Prompt Library**: Save and load your best prompts
- **Light/Dark Theme**: Accessible by default with dark mode option
- **Educational**: Examples and guides built-in

## Usage

Visit the live site or open `index.html` in your browser.

## Tech Stack

- Pure HTML, CSS, JavaScript
- No dependencies or build process
- LocalStorage for prompt library persistence

## Deployment

Deploy to Vercel with one click or host anywhere that serves static HTML.
